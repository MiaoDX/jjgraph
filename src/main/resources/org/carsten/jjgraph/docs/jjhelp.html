JJGraph user commands:

M1: Mouse button 1
M2: Mouse button 2
Ctrl-X: Press the control key while doing X.
Alt-X: Press the alt key while doing X.
 
Action Result
Ctrl-M1 on background Add node
Ctrl-M1 on node (with at least one node selected) Add edge between nodes
M1 on node/edge Selected node/edge
Shift-M1 on node Multi select nodes
Alt-M1 on node/edge Delete node/edge
M2 on node/edge Popup menu for node/edge
Drag M1 Select nodes in the specified area
